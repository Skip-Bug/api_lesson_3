# Проект: Делаем коротыши
Программа для сокращения длинных ссылок и подсчёта кликов по ним с использованием сервиса clc.li.
 - Делаем из <a href="https://github.com/Skip-Bug/api_lesson_2/tree/main" title="https://github.com/Skip-Bug/api_lesson_2/tree/main">длинных</a> ссылок <a href="clc.li/pwcjP" title="clc.li/pwcjP">короткие</a>
По коротким ссылкам ведем статистику 'кликов'   
## Что нужно сделать перед запуском
Проверьте версию Python:
   ```bash
python --version
   ```
- Должна быть версия 3.8 или выше

Установите необходимые библиотеки:
   ```bash
pip install -r requirements.txt
  ```

**Программе нужен специальный ключ (токен), чтобы работать с сервером clc.li.**
###  Получите токен
- Зарегистрируйтесь на сайте [clc.li](https://clc.li)
- Найдите там свой **API ключ** (обычно в личном кабинете или настройках профиля)
- Скопируйте его
### Создайте файл с токеном
В папке с программой создайте файл с именем **.env** (точка в начале обязательна!)
Откройте его блокнотом и напишите одну строчку:
> CLC_LI_TOKEN=вставьте_сюда_ваш_токен

## Как запустить?   
**Качаем наш код**
**Запускаем CLI** Нажмите Win + R, введите cmd
```bash
   cd укажите_путь_к_папке_api_lesson_2
```

**Запустите main.py**
```bash
python main.py -h
```
